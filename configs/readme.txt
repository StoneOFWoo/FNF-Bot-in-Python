copy the 4key.json file and modify it to your liking!
you can place your config just in the configs file itself or make another file to store it!

explanation for each of the things you will need to edit!
x <- changes the position for the x coordination
y <- same thing as x but for y you can just use for every arrow
rgb <- this is where you will be storing the color of the arrows
keybind <- this will tell the script which key it should press

extras explained:
trasparent_hold_notes <- if the hold part of the notes is transparent then set it to true, if not set it to false!
